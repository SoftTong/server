# Getting Started

## Prerequisites
- java > 11
- springframework boot > 2.5.2

## build
 - 스프링부트 gradle 빌드
   - 윈도우
        ```shell
        $ gradlew build
        ```
   - 리눅스
        ```shell
        $ ./gradlew build
        ```
## run
- 스프링부트 실행
    ```shell
    $ cd build/libs
    build/libs$ java -jar demo-0.0.1-SNAPSHOT.jar
    ```
